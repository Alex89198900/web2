Приложение CalendarTodo это глубокая доработка домашнего задания  Calendar.

Приложение CalendarTodo не адаптировано под разные экраны, только Desctop.

При клике на дату пояляетя возможность добавления расписания дел на этот день.

После добавления дел в дальнейшем при клике на эту дату, они будут отражаться в списке задач.

Есть возможность удаления и пометки как важные задания все изменения записуются в localStorage

Задачи записуются локально в localStorage где ключ это дата,а значение список дел.

Есть возможность задавать одно и тоже значение нескольким датам.

Для корректного отображения задач по каждой дате использовал функцию Map().

Функция Map() формирует уникальное значение для каждой даты calDaysMap.set(date, todoList2)
